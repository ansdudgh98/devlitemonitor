# 구름 쿠버네티스 전문가 양성과정 10기 최종 프로젝트

## 프로젝트 소개 📋
- 기존에 Opentelemetry를 사용하는 인프라 엔지니어 들을 위해 관측성이 증가된 그라파나 대시보드를 제공하기 위한 프로젝트 입니다.
- 호스팅 그라파나를 제공하여 사용자의 서버에서 그라파나 서비스를 구축하지 않아도 그라파나를 사용할 수 있도록 합니다.

## 팀 내 역활 🧑‍
- 기획
- 프론트엔트 개발
- 백엔드 개발
- AWS 인프라 설계 및 구축
- 쿠버네티스 설계 및 구축
- 그라파나 설정 설정
